Tech Stack:

Framework: Vite + React (TypeScript)
Styling: Tailwind CSS
Build Tool: Vite



Installation:

Clone the repository:
git clone https://github.com/your-username/your-repo.git
cd your-repo

Install dependencies:
npm install

Run the development server:
npm run dev



Open in browser:

http://localhost:5173
